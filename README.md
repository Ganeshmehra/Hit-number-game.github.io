Hit Game 007
This project is a fun and interactive game where the player must click on randomly generated bubbles containing numbers. The goal is to match the number in the "Hit" box to increase the score before the timer runs out. The game includes dynamic updates to the score, timer, and bubbles. It's built using HTML, CSS, and JavaScript with responsive design considerations for various screen sizes.

Features:

Dynamic Gameplay: Randomized number bubbles with a 60-second countdown timer.
Score Tracking: Score increases by 10 points for each correct match.
Responsive Design: Optimized for devices with different screen sizes using CSS media queries.
